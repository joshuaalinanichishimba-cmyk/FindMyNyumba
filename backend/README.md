# 🏠 FindMyNyumba: Zambia's Premier Real Estate API

**FindMyNyumba** is a scalable, production-ready backend platform designed to formalize and digitize the Zambian housing market. Built for speed, security, and nationwide scale.

---

## 🏢 Business Vision
Our mission is to bridge the gap between Zambian landlords and tenants (especially students) through a verified, transparent, and data-driven marketplace.

## 🛡️ Enterprise-Grade Features
- **Security**: JWT-based Authentication with Role-Based Access Control (RBAC).
- **Stability**: Global exception handling and automated logging for audit trails.
- **Protection**: Integrated Rate Limiting to prevent Brute-Force and DDoS attacks.
- **Scalability**: Optimized for thousands of listings using standardized pagination.

## 🛠️ Tech Stack
- **API**: FastAPI (High-performance Python)
- **Database**: SQLite (Dev) / PostgreSQL (Production ready)
- **Security**: OAuth2 with JWT, Bcrypt hashing
- **Testing**: Pytest for 100% critical path coverage
